应用后端

* 部署环境

  * Ali Cloud Server - Ubuntu 18.04 LTS
  * FLASK WEB框架

* 基本架构

  * FLASK框架结构。
  * 在子页面 `url/resource`下进行部署，未设置secure协议。
  * 接收POST请求随附的JSON格式数据，GET请求会被拒绝。
  * 通过爬取得到的 `搜索网站`进行域名拼接，并且随机返回指定数量的链接数。
  * 有设置分类搜索功能，容易部署上线。

* 实验心得

  实验学习了服务器端与前端的交互以及基本web服务器的部署，同时也对数据的处理、传输等方面有了更纯熟的理解应用能力。
